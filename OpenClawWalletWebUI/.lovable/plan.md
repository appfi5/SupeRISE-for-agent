

# OpenClawWallet — 托管加密钱包服务

## 项目概述
为 OpenClaw 平台打造的托管加密钱包管理前端，采用简洁极简风格设计。先构建完整的前端原型（使用 mock 数据），后续再接入后端。

---

## 页面结构

### 1. 首页（Landing Page）
- 简洁的 Hero 区域：项目名称 + 一句话介绍（"为 OpenClaw Bot 提供安全、便捷的托管钱包服务"）
- 3-4 个核心功能亮点卡片（API Key 管理、钱包创建、额度控制等）
- 行动号召按钮：「查看文档」和「进入控制台」

### 2. 文档中心（Full Docs Layout）
- 左侧边栏导航，支持多层级分组
- 文档页面包括：
  - **快速开始**：总览与核心概念
  - **安装 Skill 插件**：引导 OpenClaw 安装 wallet skill 的步骤说明
  - **配置钱包**：如何通过 API 创建和管理钱包
  - **API 参考**：API Key 使用、接口说明
  - **钱包规则**：如何设置消费额度限制

### 3. 控制台（Dashboard）
使用侧边栏布局，包含以下模块：

#### 3.1 注册 / 登录
- 简洁的登录/注册表单页面（邮箱+密码）
- 前端原型阶段使用 mock 认证

#### 3.2 账号管理
- 查看账号信息（邮箱、创建时间）
- 修改基本信息

#### 3.3 API Key 管理
- API Key 列表（名称、创建时间、状态）
- 创建新 API Key（生成后显示一次完整 key）
- 启用 / 禁用 / 删除 API Key

#### 3.4 钱包管理
- 钱包列表（地址、余额、链类型、状态）
- 创建新钱包（选择链类型）
- 查看钱包详情与交易历史
- 充值入口展示（显示充值地址和二维码占位）

#### 3.5 钱包消费规则
- 为每个钱包设置日消费限额和月消费限额
- 规则启用/禁用开关
- 当前消费额度使用进度展示（进度条）

---

## 设计风格
- 简洁极简设计（Clean & Minimal），大量留白
- 浅色主题为主
- 使用 shadcn/ui 组件库
- 控制台使用 Sidebar 布局，文档中心使用独立的 Docs Sidebar
